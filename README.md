# Generative Favelas

Original idea from [Adam Kuhn](https://codepen.io/cobra_winfrey/pen/yLvoWvb), who made it using mostly CSS and JavaScript

## Libraries used

 - GLAD (OpenGL Loader)
 - GLFW (OpenGL Framework)
 - GLM  (Maths Utilities for OpenGL)
